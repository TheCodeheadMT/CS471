How LDA Works

LDA assumes all classes share the same covariance matrix (they have the same shape/spread). Because the variance is identical, the quadratic math cancels out, resulting in a strictly linear decision boundary drawn exactly between the shifted class means.

How QDA Works

Unlike LDA, QDA calculates a separate covariance matrix for every class. This allows it to wrap around classes that have wider or tighter spreads, creating parabolic, elliptical, or hyperbolic quadratic decision boundaries.

How SVM Works

A Support Vector Machine ignores the bulk of the data clusters. It actively searches for the hyper-plane that yields the maximum margin between the absolute closest points of opposing classes (the "Support Vectors").

How Random Forest Works

Random Forest is an Ensemble of many Decision Trees. It uses Bagging (Bootstrap Aggregating) to train each tree on a random subset of data. The final classification is determined by a Majority Vote, reducing the risk of overfitting.

Ensemble Prediction:
$H(x) = \text{mode}\{h_1(x), h_2(x), \dots, h_B(x)\}$
